# cassie_axum

#### 介绍
   基于rust axum 完成web端手脚架
基础权限,用户,缓存,验证,代码生成
#####
rust axum web 是rust开发的web手脚架项目
#### 软件架构
软件架构说明
1.   rust  web框架
2.   web:axum
3.   数据库:mysql
4.   Orm:Ribatis
5.   RBAC:cabin-rs
6.   验证码:captcha
7.   数据验证:validator
